# Task 1 – Data Cleaning & Preprocessing

## Objective

To clean and preprocess raw Titanic dataset for Machine Learning.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Steps Performed

1. Imported dataset
2. Explored dataset information
3. Handled missing values
4. Encoded categorical variables
5. Standardized numerical features
6. Visualized outliers using boxplots
7. Removed outliers using IQR
8. Exported cleaned dataset

## Files Included

* Titanic-Dataset.csv
* task1_preprocessing.py
* cleaned_titanic.csv
* screenshots

## Outcome

Successfully prepared the dataset for Machine Learning by cleaning, encoding, scaling, and removing outliers.
